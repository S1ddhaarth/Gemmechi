# Gemmechi

A minimalist interface for interacting with Gemma AI locally on your device.

## 📱 Getting Started

Please keep the following in mind:

- **Model Setup**: You will be asked to select your model file upon the first launch of the application.
- **File Format**: The application exclusively supports model files in the **`.litertLM`** format.
- **Platform Support**: Gemmechi currently runs on **Android** and **Linux**.
